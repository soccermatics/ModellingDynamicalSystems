The model

The Ricker map is another discrete model of growth and regulatory feedback. It describes a variable \(x(k)\) which changes over discrete times steps. The variable might be, for example, size of an insect population.

We write the model as

\[x(k+1)) = r x(k)\exp(-x(k))\]

The parameter \(r\) determines growth rate. Investigate the dynamics of this model. Find and plot values of \(r\) where the population is stable, where it oscilLates with period 2, where it oscilLates with period 4 and where it oscilLates with period 3.
